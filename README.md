# ThreatLens AI

AI-powered cybersecurity lab for analyzing security logs and generating incident reports.

## Overview

ThreatLens AI helps security analysts and learners investigate suspicious activity by:

- Uploading security logs
- Detecting suspicious patterns
- Identifying potential attacks
- Generating incident reports
- Providing AI-powered analysis

## Planned Features

### Log Analysis
- SSH log analysis
- Apache access log analysis
- Firewall log analysis
- Windows Event Log support

### Threat Detection
- Brute-force attacks
- Suspicious IP activity
- Credential compromise indicators
- Port scanning attempts

### AI Features
- Incident summaries
- Threat explanations
- Security recommendations
- Executive reports
